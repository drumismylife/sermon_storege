# 자동 자막 오인식 교정 패턴

유튜브 한국어 자동 자막은 설교 중 발음·고유명사·성경 본문을 자주 잘못 인식한다.
아래는 지금까지 발견된 패턴이다. 새로운 패턴을 발견하면 이 목록에 추가한다.

## 일반 패턴

- 발음이 비슷한 한자어/단어로 잘못 인식 (예: "전성기" ↔ "전송기"/"찬송기")
- 성경 책 이름의 장/절 숫자 오인식 (예: "역대아" → "역대하")
- 옛스러운 표현이 현대어로 들리는 대로 잘못 인식
- 성경 본문을 읽을 때 실제 개역개정과 다른 단어로 인식되는 경우가 많음
  → **이 경우 반드시 개역개정 정확한 본문을 사용**하고, 자막의 오인식 단어는 무시한다.

## 검증 방법

1. 설교 중 성경 본문을 낭독하는 부분은 책/장/절을 먼저 식별한 뒤, 개역개정 원문과
   대조하여 자막 텍스트의 오류를 바로잡는다.
2. 인용 성경 구절 섹션(scripture-grid)에 들어가는 모든 본문은 예외 없이 개역개정
   정확한 문구를 사용한다 — 자막을 그대로 복사하지 않는다.
